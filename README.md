# BREW — コーヒー抽出ナビ

スマホ片手に「淹れて・計って・記録する」をひとつにまとめた、シングルファイルのコーヒー抽出アプリです。プロのレシピを豆量に合わせて自動換算し、内蔵タイマーで投数をガイドしながら、淹れたあとの味わいまで記録できます。

> 100% クライアントサイド（`index.html` 1 枚）で動作。データはすべてブラウザの `localStorage` に保存され、外部サーバーには一切送信されません。

## 特徴

- **10 種のプロレシピを収録** — 萩原 ／ THE COFFEESHOP、粕谷 ／ 4:6・10投式、ブルーボトルのレシピをまとめて参照。
- **豆量で自動スケーリング** — 基準レシピの注湯量・累計量・氷量を、選んだ豆量に比例して再計算。
- **抽出タイマー** — 円形ダイヤルと投数リストで「いま何 g 注ぐか／次は何秒か」をガイド。
- **挽き目記録** — 1〜10 のスライダーでグラインダー設定を記録。
- **豆の在庫管理** — ロースター・原産地・焙煎度・焙煎日（焙煎後日数を自動計算）・精製・フレーバーノートを登録。使用中／使い切りで管理。
- **テイスティング記録** — 酸味・甘み・ボディ・余韻（各 5 段階）、総合★評価、フレーバーメモ、「また淹れたい」フラグを保存。
- **オフライン対応・登録不要** — インストールもアカウントも不要。ブラウザで開くだけ。

## 使い方

1. `index.html` をブラウザで開く（PC・スマホどちらも可）。
2. **淹れる** タブでレシピを選ぶ。
3. **設定** 画面で豆量を調整し、必要なら挽き目・使う豆を選択。レシピが自動換算される。
4. **タイマー開始** → ダイヤルとガイドに沿って抽出。
5. **淹れ終わり → 記録** で味わいを評価して保存。
6. **記録** タブで過去の抽出を、**豆** タブで在庫を確認。

### ローカルで開く

```bash
git clone https://github.com/slothfront/coffee-log.git
cd coffee-log
# そのままブラウザで開くか、簡易サーバーで配信
python3 -m http.server 8000   # → http://localhost:8000
```

## 技術メモ

- 依存ライブラリなし。HTML / CSS / バニラ JavaScript のみの単一ファイル。
- フォントのみ Google Fonts（Archivo / Space Grotesk）を CDN から読み込み。
- 保存キー：豆は `coffee_beans`、抽出記録は `coffee_brews`（いずれも `localStorage`）。
- データはブラウザ・端末ごとにローカル保存。キャッシュ削除や別端末では引き継がれません。
